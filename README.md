# Описание
Pineapple Price List (PPL) - это сайт типа имиджборд. Сайт позволит пользователям создавать свои доски и треды. Анонимность пользователей обеспечивает регистрация по имени-паролю, никакой дополнительной информации. Сайт позволяет прекреплять к постам файлы любого типа, а также поддерживает цепочки ответов.

# Техническое задание
### Обзор проекта
При заходе на сайт пользователю предлогается выбрать доску. Незарегистрированный пользователь может читать все доски и треды, но для создания собственных - требуется регистрация. Регистрация происходит по логину+паролю.
### Реализация постов
Каждый пост имеет уникальный id и является ответом на другой пост. Пост состоит из текста (обязательно), картинки и прикреплённого файла (опционально).
### Реализация БД
БД состоит из трёх таблиц: __users__, __boards__, __posts__.
