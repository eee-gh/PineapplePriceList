# Описание
Pineapple Price List (PPL) - это сайт типа имиджборд. Сайт позволит пользователям создавать свои доски и треды. Анонимность пользователей обеспечивает регистрация по имени-паролю, никакой дополнительной информации. Сайт позволяет прекреплять к постам файлы любого типа, а также поддерживает цепочки ответов.
